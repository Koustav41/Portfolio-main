## Content & Case Study Guidelines

A great portfolio keeps the focus on impact. For every project include:

- Project title and role
- Timeframe and collaborators
- Problem statement and goals
- Approach (design & technical decisions)
- Outcome — measurable results, screenshots, links
- Key takeaways and next steps

---

## Performance, Accessibility & SEO

- Use semantic HTML and ARIA where needed
- Optimize and lazy-load images (use srcset / modern formats)
- Ensure keyboard navigation and color contrast
- Add structured data (JSON-LD) for rich results
- Configure meta tags and Open Graph images for sharing
